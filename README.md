## Use script

1. Install node
2. Run `npm i`
3. Clone [Protobuf](git@github.com:protocolbuffers/protobuf.git) to disk
4. Open `ProtobufForUE.js`
5. Alter `protobufRepoDir` to your protobuf repo path
6. Run `node ProtobufForUE.js`
